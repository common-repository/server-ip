Server IP - Simply displays IP Address (and hostname) on the WordPress dashboard admin panel or on a public page.

== Description ==

Displays IP Address and hostname on the WordPress dashboard admin panel.
Also permit to show server ip address on a page, on a post or on sidebar widget.

Knowing the IP address of your website or server may be useful in certain situations but you can’t find IP address in the WordPress admin panel. So, how to find IP address of WordPress website ?
There is a quick solution : Server IP plugin.

This plugin simply display IP Address on the WordPress dashboard admin panel or everywhere on public page, on a sidebar widget with a shortcode.

Very simple to install, simple to use.


== Installation ==

1. Upload `server-ip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is Server IP free of charge ? =

Yes, of course ! For ever.

= How to use this plugin properly ? =

Two ways :

1. Look for the server address IP in the admin dashboard.

2. To show it in your page or post use this shortcode  :

[server_ip]

= Can I show Server IP Address in a sidebar widget ? =

Yes, it works in a "text widget" with this same shortcode [server_ip].

= Where is the plugin settings ? =

There is no settings.

= Can I show User IP Address or Admin IP Address with this plugin ? =

No, you can't. This plugin displays only server IP Address.
To display User IP Address (visitor IP), use this WordPress plugin :

http://wordpress.org/plugins/user-ip/

To display Admin IP Address (your IP), use this WordPress plugin :

http://wordpress.org/plugins/admin-ip/
